# 🌤️ Previsão do Tempo

Aplicação web que exibe o clima atual e a previsão para os próximos 5 dias de qualquer cidade do mundo.

## 🔗 Demo ao vivo

[Clique aqui para ver o projeto funcionando](https://brunotenorio441.github.io/previsao-do-tempo)

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- API: OpenWeatherMap

## ✨ Funcionalidades

- Busca de clima por nome de cidade
- Temperatura atual, sensação térmica, umidade, vento e visibilidade
- Previsão para os próximos 5 dias
- Layout responsivo
